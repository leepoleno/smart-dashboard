# Smart Dashboard ✨

PWA приложение для управления задачами, заметками и отслеживания прогресса.

## Особенности

- 📋 Управление задачами и привычками
- 📝 Создание заметок
- 📊 Отслеживание прогресса
- 🦊 Маскот-мотиватор
- 🔥 Система стриков
- 📱 PWA для установки на устройство
- 🎨 Нежный и приятный дизайн
